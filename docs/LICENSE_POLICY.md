# Landau Fund License Policy

Status: active from 2026-05-24.

## Public Non-Code Materials

Public Landau Fund non-code materials are released under:

Creative Commons Attribution 4.0 International (CC BY 4.0)

Official license:
https://creativecommons.org/licenses/by/4.0/

This applies to:

— texts;
— research materials;
— manifestos;
— methodologies;
— tables;
— diagrams;
— publications;
— public laboratory reports;
— public documentation.

## Required Attribution

Any use must include:

— attribution to Landau Fund;
— a link to the original public GitHub repository;
— a link to the CC BY 4.0 license;
— an indication of changes when changes were made.

Suggested attribution:

Based on materials by Landau Fund, Antigravity Open Research Initiative, licensed under CC BY 4.0:
https://creativecommons.org/licenses/by/4.0/

Original repository:
https://github.com/LANDAUFUND/landau-fund

## Private Materials

Private repositories are closed internal workspaces.

Unreleased private materials do not receive a public reuse license merely because they exist in GitHub.

Before private materials become public, they must pass publication review and be copied into a public repository with the correct license notice.

## Code and Software

If software source code, simulations, AI models, or engineering systems are added later, they may need a software license such as MIT, Apache 2.0, or GPL.

Until a separate software license is explicitly added, CC BY 4.0 should be treated as the license for public non-code research materials only.
